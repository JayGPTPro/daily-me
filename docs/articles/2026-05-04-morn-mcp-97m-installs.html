AI יום שני, 4 במאי 2026

Model Context Protocol עבר 97 מיליון התקנות. הופך לתשתית AI

Model Context Protocol (MCP) של אנתרופיק עבר ציון דרך משמעותי במרץ 2026. הפרוטוקול עבר 97 מיליון התקנות.

MCP הוא פרוטוקול פתוח שמאפשר ל-AI לתקשר עם כלים, מסדי נתונים ומערכות. הוא הוצע במאי 2025. בתחילה, אומץ רק על ידי קומץ סטארטאפים. כעת הוא הופך לתשתית בסיסית.

כל ספקיות ה-AI הגדולות שולחות כלים תואמי MCP. גוגל, OpenAI, מטא, Mistral - כולן הצטרפו. אפילו הסוכנים החדשים של xAI (Grok 5) תומכים ב-MCP.

למה זה חשוב? בלי פרוטוקול אחיד, כל סוכן AI היה צריך אינטגרציה ייחודית עם כל כלי. MCP פותר את הבעיה. דומה למה ש-USB עשה לחומרה.

אצל המפתחים והשוק - תאוצה חזקה. ב-Cursor IDE לבד יש מעל 10,000 שרתי MCP מותקנים. Claude Code (גרסת Terminal של אנתרופיק) הפכה לפלטפורמה הפופולרית ביותר לסוכני MCP.
