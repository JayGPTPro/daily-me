Mistral משחררת מודל דגל 128B חדש ומצב Work אגנטי ב-Le Chat

Mistral AI. החברה הצרפתית שהפכה לאחת ממובילות ה-AI באירופה. הודיעה השבוע על השקת מודל דגל חדש בגודל 128 מיליארד פרמטרים. המודל החדש מתחרה ישירות עם GPT-5.4 ועם Claude Opus 4.7.

בנוסף למודל. Mistral השיקה את מצב Work ב-Le Chat. מצב אגנטי שמאפשר למודל לבצע משימות מורכבות בעצמו. כולל קידוד אסינכרוני בענן. גלישה. שליחת מיילים וניתוח נתונים.

הסשנים האסינכרוניים מבוססי הענן הם החידוש המשמעותי ביותר. המשתמש יכול לשלוח משימת קידוד מורכבת. לסגור את הדפדפן. ולחזור שעות אחר כך כשהמודל סיים את העבודה. דומה ל-OpenAI Codex.

המודל זמין דרך API במחיר של 1.50 דולר לכל מיליון טוקני קלט ו-7.50 דולר לפלט. זול יותר מהמתחרים האמריקנים. מצב Work זמין בתוכנית Pro של Le Chat ב-30 דולר לחודש.

Mistral מתבססת בראשי-טרמין על השוק האירופי. וטוענת שהיא ההצעה היחידה ה-AI ב״סוורניות אירופית״. זוכה לתמיכת ממשלות צרפת. גרמניה ואיטליה.
