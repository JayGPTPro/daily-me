AWS ו-OpenAI: מודלי GPT האחרונים מגיעים ל-Bedrock. גם Codex ו-Managed Agents

באירוע ״What's Next with AWS 2026״ הודיעו אמזון ו-OpenAI על שותפות מורחבת ופתיחת מסלול חדש. המודלים החדשים של OpenAI יהיו זמינים מיידית באמזון Bedrock - הפלטפורמה של AWS למודלי AI.

ההשקה כוללת את GPT-5.5 (המודל הכי חדש של OpenAI). את גרסת ה-Codex החדשה לקידוד. ואת Amazon Bedrock Managed Agents - שירות חדש לבניית סוכני AI מבוססי OpenAI עם ניהול אוטומטי של הזיכרון. הכלים והאינטגרציות.

המהלך משמעותי כי עד עכשיו OpenAI היה זמין בעיקר דרך Microsoft Azure. כעת AWS מצטרפת. מה שמרחיב את הזמינות ומקטין את התלות של OpenAI ב-Microsoft.

המחירים זהים לאלו של OpenAI ישירות. אך AWS מציע יתרונות בתשתיות ובאינטגרציה עם שאר שירותי הענן. לקוחות AWS יכולים לבחור עכשיו בין Anthropic. Mistral. Meta ו-OpenAI ישירות מתוך Bedrock.

סלמון לון. סמנכ״ל AI ב-AWS. אמר: ״הלקוחות שלנו רצו את OpenAI. עכשיו הם יקבלו אותם עם כל היתרונות של Bedrock״. הצפי הוא שהמהלך יקצין את התחרות בין AWS ל-Microsoft Azure על שוק ה-AI הארגוני.
