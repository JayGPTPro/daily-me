🤖 אנתרופיק 4 באפריל 2026

קוד המקור של Claude Code דלף דרך npm - 513,000 שורות קוד הופצו ב-GitHub

ב-31 במרץ 2026, אנתרופיק דלפה בשגגה את קוד המקור המלא של Claude Code - כלי הפיתוח הפופולרי שלה. הדליפה התרחשה דרך חבילת npm גרסה 2.1.88: קובץ source map של 59.8MB שהכיל 513,000 שורות TypeScript לא מוצפן.

איך זה קרה? קובץ debug פנימי נכלל בטעות בעדכון שגרתי. הקובץ הפנה לארכיב ZIP שמאוחסן בתשתית Cloudflare R2 של אנתרופיק עצמה - ובו הקוד המלא ב-1,906 קבצים. מדובר בשגיאת .npmignore בסיסית: הקובץ לא הוחרג מהחבילה הציבורית.

תוך שעות, המפתח קהילת הפיתוח הגיבה: הארכיב הורד, הועלה ל-GitHub, ופוצל (forked) עשרות אלפי פעמים. מפתחים מכל העולם ניתחו, פיתחו גרסאות ב-Rust ו-Python, ושיתפו ממצאים ברשתות החברתיות. "Great Claude Code Leak of 2026" הפך לתגית ויראלית.

אנתרופיק הגיבה בהצהרה רשמית: "מדובר בשגיאת אריזה שנגרמה מגורם אנושי, לא פריצת אבטחה. לא נחשפו נתוני לקוחות או פרטי כניסה." החברה מתקינה כעת בדיקות למניעת אירוע דומה. ה-source map הוסר מהשרתים.

ההשלכות? מחד, אנתרופיק מאבדת יתרון קנייני על מתחרים שיכולים ללמוד מהארכיטקטורה. מאידך, חלק מהמפתחים טוענים שהדבר יסייע לאקוסיסטם הפתוח ויחזק את האמון בכלי. לשאלה "האם זה פרילנסר של אפריל?", Yahoo Tech ענה רשמית: "לא - זה אמיתי".
