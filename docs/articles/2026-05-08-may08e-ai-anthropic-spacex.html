אנתרופיק הכפילה את מגבלות Claude Code אחרי שותפות עם SpaceX

אנתרופיק הודיעה אמש (7 במאי) על הכפלת מגבלות Claude Code עבור מנויי Pro, Max, Team ו-Enterprise. ההכפלה היא תוצאה ישירה של שותפות חדשה עם SpaceX. שדרגה משמעותית לכל מי שמשתמש בכלי הקודינג של אנתרופיק.

במסגרת השותפות, אנתרופיק מקבלת גישה למרכז הנתונים Colossus One של SpaceX, שמספק יותר מ-300 מגה-וואט של קיבולת חישוב חדשה. שווה ערך ליותר מ-220,000 GPUs של Nvidia. זה אחד הקפיצות הגדולות בקיבולת חישוב של חברת AI אי פעם.

מבחינה פרקטית: מנויי Pro ו-Max קיבלו הסרה מלאה של "peak hour restrictions". הם יכולים להשתמש ב-Claude Code בלי הגבלת שעות שיא. מגבלות 5 השעות הוכפלו, מה שאומר פי 2 משימות בכל חלון.

הצעד מסמן עד כמה אנתרופיק מתבצרת בקטגוריית הקודינג. מתחרים כמו Cursor ו-Copilot של GitHub כבר הרגישו את הלחץ. עכשיו ההפרש מתרחב.

גם מבחינת אסטרטגיית התשתיות: שותפות עם SpaceX (שמחזיקה ב-xAI דרך אילון מאסק. מתחרה ישיר של אנתרופיק) היא הימור מעניין. שני הצדדים, ככל הנראה, מבינים שיש מספיק "עוגה" לכולם.
