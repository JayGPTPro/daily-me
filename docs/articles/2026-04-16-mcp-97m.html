🤖 AI יום חמישי, 16 באפריל 2026 · מהדורת ערב

פרוטוקול MCP של אנתרופיק: 97 מיליון התקנות — כל ספקיות ה-AI הגדולות אימצו אותו כסטנדרט

Model Context Protocol (MCP) שפותח על ידי אנתרופיק, חצה ב-מרץ 2026 את רף 97 מיליון ההתקנות — ומחזיק בתואר הפרוטוקול הנפוץ ביותר לחיבור כלים חיצוניים למודלי שפה גדולים (LLMs).

מה זה בעצם MCP? בפשטות: זה "שקע חשמל" אחיד שמאפשר לכל כלי (מסד נתונים, API, קובץ, אפליקציה) להתחבר לכל מודל AI. לפני MCP, כל ספק AI פיתח שיטת חיבור משלו — מה שיצר פיצול וכאב ראש למפתחים.

המשמעות: כל מי שבנה שרת MCP פעם אחת — יכול לחבר אותו לקלוד, ל-GPT-5, ל-Gemini ול-Grok ללא שינויים. זה הגדיל דרמטית את כמות הכלים הזמינים לכל מודל AI.

נכון לאפריל 2026: OpenAI, Google, xAI (Grok), Mistral, Cohere — כולן שולחות תמיכת MCP בפלטפורמות שלהן. NVIDIA הכריזה שכל מסגרות ה-agentic שלה תומכות ב-MCP כברירת מחדל.

בנוסף, כל ספקיות הענן הגדולות (AWS, Azure, GCP) בנו שכבות MCP לשירותיהן — מה שהפך את הפרוטוקול לתשתית AI ממש כפי שהתקנים כמו HTTP הפכו לתשתית האינטרנט.

האירוניה: אנתרופיק שמרה את הפרוטוקול פתוח ולא מנסה להרוויח ממנו ישירות — בדיוק כמו שטים ברנרס-לי לא הרוויח ישירות מ-HTTP. אך ה-"מוניטין" שמגיע מלהיות "האבא של MCP" מחזק את מעמד אנתרופיק בתעשייה.
