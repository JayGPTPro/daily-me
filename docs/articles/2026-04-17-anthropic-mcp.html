Anthropic MCP: 97 מיליון התקנות, כל ספקי ה-AI אימצו

Model Context Protocol (MCP) של Anthropic חצה 97 מיליון התקנות במרץ 2026, מה שהופך אותו לאחד הפרוטוקולים שאומצו במהירות הגדולה ביותר בהיסטוריה של פיתוח תוכנה.

MCP הוא תקן פתוח שמאפשר למודלי AI להתחבר לכלים חיצוניים, מסדי נתונים ושירותים באופן אחיד. במקום שכל חברת AI תכתוב את חיבוריה הייחודיים לכל שירות, MCP יוצר שפה משותפת.

מה שמרשים במיוחד הוא שהמתחרים הישירים של Anthropic, כולל OpenAI, Google ו-xAI, כבר שלחו כלים תואמי MCP. זהו מקרה נדיר בתעשיית ה-AI שבו תקן של חברה אחת מאומץ על ידי כולן.

בפועל, MCP מאפשר לסוכני AI כמו Claude Code (שאיתו ג'יי עובד) להתחבר בקלות ל-Slack, Notion, GitHub, Supabase ועשרות כלים נוספים. מפתחים יכולים לכתוב שרת MCP אחד שעובד עם כל מודל AI.

Dario Amodei, מנכ"ל Anthropic, אמר שהם מתייחסים ל-MCP כ"מתנה לאקוסיסטם" ולא כנכס קנייני. "רצינו שה-AI יוכל להתחבר לעולם, ולא שהעולם יהיה תלוי בנו לצורך זה."
