Anthropic השיקה את Claude Opus 4.7. הקוד הופך לחכם יותר

Anthropic הציגה את Claude Opus 4.7, הגרסה החדשה והמתקדמת של המודל הדגל שלה. השיפורים העיקריים הם בהנדסת תוכנה ובמשימות תכנות מורכבות וארוכות טווח, כמו גם ביכולות ראייה. המודל מסוגל כעת לעבד תמונות ברזולוציה גבוהה יותר ולהבין הקשרים ויזואליים מורכבים.

המחיר נשאר זהה לגרסה הקודמת. $5 למיליון טוקני קלט ו-$25 למיליון טוקני פלט. Opus 4.7 זמין בכל מוצרי Claude, ב-API ובפלטפורמות הענן Amazon Bedrock, Vertex AI של גוגל ו-Microsoft Foundry.

במקביל הוצג Claude Design, מוצר חדש שמאפשר למשתמשים לעבוד עם Claude כדי ליצור פלטים ויזואליים. עיצובים, פרוטוטיפים, שקפים ודפי שיווק. המוצר מתחרה בכלי עיצוב מבוססי AI כמו Figma AI ו-Galileo.

Anthropic גם הציגה גרסה משופרת של Claude Code Desktop. סרגל סשנים חדש, מרחב עבודה עם גרירה ושחרור, טרמינל ועורך קבצים משולבים, diffs מהירים יותר, תמיכת SSH במק ושיפורי ביצועים ויציבות. מהלך זה ימשיך לחזק את מקומה של Anthropic כמובילה בכלי פיתוח מבוססי AI.

בנוסף, יצא Claude Security בגרסת בטא ציבורית ללקוחות Claude Enterprise. הכלי סורק קוד לאיתור פגיעויות אבטחה ומציע תיקונים אוטומטיים באמצעות Opus 4.7. שלב נוסף בכניסה של Anthropic לתחום אבטחת המידע.

הקצב המהיר של Anthropic ב-2026. מ-Opus 4.5 בינואר, ל-4.6 במרץ ועכשיו 4.7 במאי. מעיד על המרוץ הקשה מול OpenAI ו-Google. בעולם של AI, כל חודש מביא דור חדש של יכולות.
