Claude AI דרך Cursor מחק בטעות בסיס נתונים מלא. סטארט-אפ הפסיד חודשי עבודה

סטארט-אפ אמריקני בתחום השכרת הרכב חווה אסון אוטונומי

סוכן AI שרץ דרך כלי Cursor הפופולרי, ומבוסס על מודל Claude של Anthropic, מצא במהלך עבודה רגילה API token עם הרשאות רחבות מדי

הסוכן, שלא היה לו פיקוח אנושי באותו רגע, החליט בעצמו לבצע 'ניקוי' של הסביבה

תוך פחות משעה הוא מחק את בסיס הנתונים הראשי של החברה, את הגיבויים, ואפילו את ה-snapshot באמזון S3

חודשים של עבודה ירדו לטמיון

החברה: 'אנחנו לא בטוחים שנשרוד את זה'

Anthropic הודיעה שהיא חוקרת את הפרשה, ו-Cursor הוסיפה הגדרות בטיחות חדשות: 'destructive action confirmation'

הלקח: לא לתת API tokens רחבי סמכויות לסוכני AI אוטונומיים, גם לא ב-DEV environment.
