Anthropic מרחיבה את פלטפורמת Claude עם Managed Agents

Anthropic הודיעה היום על שורת עדכונים משמעותית לפלטפורמת המפתחים שלה. החברה מרחיבה את היכולות של Claude עם דור חדש של Managed Agents שכולל חלימה (dreaming), אורקסטרציה רב-סוכנית, ניהול outcomes ו-webhooks.

בנוסף, החברה השיקה גרסת בטא ציבורית של self-hosted sandboxes, שמאפשרת לארגונים להחזיק את הקבצים, החבילות והשירותים הרגישים בתשתית שלהם או אצל ספק sandbox מנוהל, בעוד שלולאת הסוכן שמטפלת באורקסטרציה, ניהול הקשר וטיפול בשגיאות נשארת בתשתית של Anthropic.

Claude Code קיבל תובנות שימוש עשירות יותר, גלילת diffs ידידותית למקלדת ורנדור משופר של רשימות משימות במארקדאון. במקביל נוספה הגדרה ארגונית עבור Claude.ai cloud MCP connectors.

המהלך הזה מאותת על מאמץ של Anthropic לחזק את האחיזה שלה בשוק הסוכנים הארגוניים, אחרי שב-KPMG נפרסה Claude ל-276,000 עובדים ב-138 מדינות.
