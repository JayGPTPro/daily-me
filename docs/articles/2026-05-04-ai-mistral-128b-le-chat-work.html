Mistral חושפת מודל 128B חדש ומצב Work אגנטי ב-Le Chat

Mistral AI הצרפתית הציגה אמש את מודל הדגל החדש שלה. בגודל 128 מיליארד פרמטרים. עם תמיכה בסשנים אסינכרוניים מבוססי ענן לקידוד. ובמצב Work חדש ב-Le Chat שמספק יכולות אגנטיות מלאות.

המודל החדש מציע ביצועים תחרותיים מול Claude 4.7 ו-GPT-5.5 במשימות קוד. אך עם עלות מופחתת ב-60%. במבחני HumanEval ו-SWE-bench. הוא הגיע ל-93% ו-67% בהתאמה.

מצב Work החדש מאפשר למודל לבצע משימות מורכבות מקצה לקצה ללא התערבות. כולל גישה לאינטרנט. הרצת קוד. ניתוח קבצים ושימוש בכלים אישיים שהמשתמש מגדיר. ראינו דמו שבו הוא בנה אפליקציה שלמה ב-2 שעות.

הסשנים האסינכרוניים מאפשרים למפתחים לפתוח משימה. לסגור את המחשב. ולחזור מאוחר יותר לקבל את התוצאות. המערכת ממשיכה לעבוד בענן. זה דומה ל-Background mode של GPT-5.5 אך מתקדם יותר.

Mistral מציעה את המודל החדש דרך ה-API שלה. ובמסך הצרכן Le Chat. המנוי החודשי שדרגה את עצמה ל-25 אירו. וכוללת גישה לכל היכולות החדשות. ללא הגבלה.

המהלך מחזק את המעמד של Mistral כשחקנית הצרפתית הראשונה ב-AI. החברה גייסה השבוע 2 מיליארד אירו בסבב מימון נוסף. וכעת שווה 25 מיליארד אירו. אבן דרך לאקוסיסטמת ה-AI האירופאית.
