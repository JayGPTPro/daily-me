Claude Opus 4.8 שובר שיא ב-SWE-bench עם 88.6%

אנת'רופיק שיחררה את Claude Opus 4.8, גרסה שמשבחת את הביצועים של המודל הגדול ביותר שלה במספר בנצ'מרקים מרכזיים. במבחן SWE-bench Verified, מבחן שנחשב סטנדרט תעשייתי למשימות תכנות אמיתיות, המודל הגיע ל-88.6%, שיא חדש.

ב-Terminal-Bench 2.1, מבחן שמודד יכולת לבצע משימות אמיתיות בקו פקודה, Opus 4.8 הגיע ל-74.6%. המספרים האלה מציבים את אנת'רופיק במקום הראשון בשני הבנצ'מרקים החשובים ביותר למפתחים ב-2026.

חדשנות נוספת: ה-workflows של parallel subagents מאפשרים למודל להפעיל מספר משימות במקביל ולסנכרן את התוצאות. זה הופך את העבודה עם פרויקטים גדולים למהירה משמעותית, במיוחד באוטומציות מורכבות.

המחיר נשאר זהה. 5 דולר לקלט ו-25 דולר לפלט למיליון טוקנים, כולל מצב "פי 2.5 מהירות" שמאפשר השלמות מהירות יותר ללא תוספת תשלום. החלטה מודעת של אנת'רופיק לא להעלות מחיר למרות שיפור משמעותי בביצועים.

מה זה אומר על Claude Code: רוב המשתמשים של Claude Code יראו שיפור מורגש במשימות מורכבות. אם אתה עובד עם projects רב-קבציים, ניתוח קוד גדול, או agents אוטונומיים. השדרוג ל-4.8 הוא חובה.

המרוץ ב-AI הופך לאתגר. אנת'רופיק מעדיפה איכות וביצועים, גוגל מעדיפה מהירות ומחיר. ב-2026 ההבדל הזה מגדיר את הבחירות של מפתחים, כשכל פרויקט בוחר ספק לפי הצרכים המהותיים שלו.
