אנתרופיק: Claude Opus 4.7 זמין. מודל הזרז החדש לעבודה מקצועית

אנתרופיק שחררה רשמית את Claude Opus 4.7. מודל ה-frontier החדש של החברה. למשתמשי כל המוצרים: Claude Web. Claude Code. ו-API. המחיר נשאר אותו דבר כמו Opus 4.6: $5 לקלט (1M tokens). $25 לפלט.

שיפורים מרכזיים:

• עבודה אגנטית: שיפור 28% בבנצ'מרקים של ביצוע משימות מורכבות
• קודינג: 15% בלעדים נוספים על Claude Code
• ייצוג קוד: יכולת חדשה להפיק UI מתאר טקסט
• בטיחות: התקדמות במניעת hallucinations

אנשי הפיתוח (Anthropic Engineering): "Opus 4.7 הוא הצעד הגדול ביותר שעשינו בעבודה אגנטית מאז 4.0".

שחרורים נלווים השבוע:

• Managed Agents - שירות חדש להפעלת סוכני AI לטווח ארוך עם ממשק יציב. סנדבוקסים בטוחים. הפעלה מהירה. מטרה: לאפשר אוטומציה של תהליכים שלמים
• Claude Security בבטא - לקוחות Claude Enterprise יכולים לסרוק קוד לפגיעויות ולקבל תיקונים מ-Opus 4.7
• Claude Code עדכון רחב: בחירת מודל חכמה. כלי ניקוי פרויקט. הרשאות מוגברות. OAuth משופר. תמיכה ב-Windows ו-PowerShell
• מחברי יישומים יצירתיים: Adobe. Blender. Ableton. Affinity. Autodesk Fusion. ניתן עכשיו לתפעל את היישומים האלה ישירות מ-Claude
• שינויי API: חלון 1M טוקנים נסגר ב-Sonnet 4.5/4. עברו ל-Sonnet 4.6 או Opus 4.6 לקבל את החלון הגדול

מחברי היישומים היצירתיים מחללים את ה-Workflow של מעצבים. אנימטורים. מוזיקאים. וארכיטקטים. במקום לשחק במידע בין יישומים. אפשר לבקש מ-Claude לבצע פעולות ישירות בכל יישום נתמך.

בקהילת המפתחים: התגובות חיוביות מאוד. בעיקר על Managed Agents שצפוי להפוך את הבניית סוכני AI הרבה יותר פשוטה. וגם על Claude Security ש"חוסך 80% מזמן הסקירת קוד".
