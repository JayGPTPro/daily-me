Microsoft Build 2026: סדרת MAI

בכנס Build 2026 הציגה Microsoft חבילת מודלי MAI מקוריים, כולל מודל קוד ל-GitHub Copilot ומודלים מתמחים לתמלול, חשיבה, דיבור ותמונות.

בנוסף הוצגו Windows Agent Framework APIs לסוכני AI אוטונומיים במערכת ההפעלה. המפתחים יוכלו לבנות סוכנים שירוצו בלי התערבות אנושית.

Copilot Agent Mode מציע זרימות קוד רב-שלביות ב-GitHub. כעת ניתן להאציל למשל "הוסף תכונה X" וה-Agent יבנה את כל ה-PR.

עדכוני Azure AI Foundry רב-מודליים: עכשיו ניתן לעבוד עם מודלים של OpenAI, Anthropic, Meta ו-Microsoft בממשק אחד.

סטיב באלמר בלם בכנס. הציג את החזון העתידי של Microsoft AI לעשור הבא.

למפתחים: כל הכלים זמינים בייצור. תהליך ההגירה מ-Code Copilot הקודם פשוט יחסית.
