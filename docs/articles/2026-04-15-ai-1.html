Anthropic משיקה את Claude Code כמוצר עצמאי

Anthropic השיקה רשמית את Claude Code כמוצר עצמאי נפרד, ולא רק כתוסף לעורכי קוד קיימים. הסוכן החדש פועל ישירות מהטרמינל ומסוגל לנהל מחזורי פיתוח שלמים ללא התערבות אנושית מתמדת. זהו צעד אסטרטגי של Anthropic בתחרות ההולכת ומתחזקת על שוק כלי הפיתוח מבוסס AI.

הסוכן יכול לשכפל מאגרי קוד, לכתוב ולהריץ טסטים אוטומטיים, לתקן pipelines שנכשלו ולפתוח pull requests אוטומטית ב-GitHub ובמערכות CI/CD נוספות. היכולת לנהל את כל מחזור החיים של הקוד היא מה שמבדיל את Claude Code מכלים אחרים כמו GitHub Copilot.

המוצר מופעל על Claude Sonnet 4.6 כמודל ברירת המחדל, עם האפשרות לשדרג ל-Opus 4.6 למשימות מורכבות במיוחד. Anthropic הדגישה כי Sonnet 4.6 מספיק לרוב משימות הפיתוח היומיומיות, בעוד Opus 4.6 שמור לבעיות אדריכלות מורכבות ובדיקות עומק.

בהשוואה לכלים מתחרים כמו Devin של Cognition AI ו-Cursor, Claude Code מציע אינטגרציה מעמיקה יותר עם מודלי Claude ותמיכה רחבה יותר בשפות תכנות ופריימוורקים. המוצר כולל גם תמיכה ב-multi-repo workflows, מה שמאפשר לסוכן לעבוד עם מספר מאגרים בו-זמנית.

הדגמות ראשוניות מראות כי Claude Code מסוגל לקחת תיאור של באג ולהשלים את כל תהליך התיקון בפחות מ-10 דקות, כולל כתיבת הקוד, בדיקתו, תיקון שגיאות ופתיחת PR. זהו שיפור משמעותי לעומת הגרסאות הקודמות שדרשו יותר הנחיות ותיקונים ידניים.

Anthropic מתכננת להרחיב את יכולות Claude Code בחודשים הקרובים, כולל תמיכה בפריסה אוטומטית ל-Vercel, AWS ו-Google Cloud. התמחור עדיין לא הוכרז לגמרי, אך הצפי הוא מודל מנוי חודשי עם מכסות שימוש בהתאם לגודל הארגון.
