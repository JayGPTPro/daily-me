Anthropic מחזירה תמיכה בכלי-קוד אבל מתחת למד נפרד

חברת Anthropic הודיעה ביום חמישי כי היא מחזירה תמיכה בכלי סוכן (agents) חיצוניים במנויי קלוד בתשלום, אך מחזירה אותם תחת מד נפרד של credits.

החל מ-15 ביוני, השימוש בקלוד דרך API ייפרד מהמנוי הסטנדרטי לצ'אט. מנויי Pro יקבלו $20 בקרדיטים, מנויי Max 5x יקבלו $100, ומנויי Max 20x יקבלו $200 לחודש.

המהלך נועד להתמודד עם הצריכה האקסטרימית של agents שיכולים לשרוף משאבי חישוב הרבה יותר מהר ממשתמש אנושי. המודל 'all-you-can-eat' לא שורד את עידן ה-agents, לדברי מומחים.

במקביל, חברת OpenAI הודיעה על שיווק חדש: שני חודשי שימוש חינם ב-Codex ללקוחות עסקיים חדשים. ה-CEO סם אלטמן פרסם זאת ב-X.

ל-Jay זה רלוונטי: שימוש מקצועי בקלוד דרך Claude Code יהיה תחת המד החדש. כדאי לתכנן את שימוש ה-API מראש כדי לא להתבזבז בקרדיטים. גם הגדלת המגבלות השבועיות בקלוד קוד ב-50% נכנסה לתוקף עד 13 ביולי 2026.
