Gemini 3.1 Ultra: 2 מיליון טוקנים, ריצת קוד בתוך השיחה

גוגל הציגה את Gemini 3.1 Ultra, הדגם החדש של משפחת ג'מיני. זה השדרוג המשמעותי ביותר השנה, ומשיב לתחרות החריפה מצד OpenAI ואנת'רופיק.

הפיצ'ר הבולט: חלון הקשר של 2 מיליון טוקנים שעובד בצורה מקורית על טקסט, תמונה, אודיו ווידאו - בלי שלב של תרגום (transcription) באמצע. כלומר, הדגם רואה תמונה ומבין אותה ישירות, בלי לתרגם אותה תחילה לטקסט.

תוסף Code Execution חדש: הדגם יכול עכשיו לכתוב קוד, להריץ אותו בסביבת sandbox מבודדת, לראות את התוצאה, ולתקן באג ים תוך כדי שיחה. זה מקרב את ג'מיני לסטנדרט של Claude Code ושל ChatGPT עם Code Interpreter.

שיפור Grounding: גוגל טוענת שהזיות (hallucinations) ירדו ב-37% בהשוואה לג'מיני 3.0 Ultra, בזכות אינטגרציה הדוקה עם חיפוש Google ועם מסדי נתונים שונים.

מחיר: 15 דולר לקלט מיליון טוקנים, 60 דולר לפלט. קצת זול יותר מ-GPT-5.5, אבל לא הזול ביותר בשוק. ההגעה ל-API היא מהיום, ובמקביל פותחים את התכונה ל-Gemini Advanced.

ראשי שותפויות נחתמו: סמסונג מתחייבת ל-800 מיליון מכשירי מובייל עם ג'מיני עד סוף 2026. דה-לוקס לפני שנה זה היה 400 מיליון.
