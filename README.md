# Daily Me

**My world, every day.**

Your personal daily news dashboard that updates twice a day — morning and evening. Built with Claude Code, hosted on GitHub Pages.

## What is Daily Me?

Daily Me is an AI-powered personal newspaper. It collects, translates, and summarizes the news that matters to YOU — in Hebrew, in a beautiful dashboard format, delivered fresh every morning and evening.

**No noise. No irrelevant stories. Just what you care about.**

## Features

- **Morning & Evening editions** — Different focus for each (morning: what happened overnight + today's schedule; evening: day recap + tomorrow preview)
- **Weather + clothing recommendation** — "Sweater and umbrella" not just "16 degrees"
- **Dollar exchange rate** — With sparkline chart showing weekly trend
- **Calendar integration** — Israeli holidays + Google Calendar meetings
- **AI News** — Major AI stories, translated and summarized to Hebrew
- **Israeli Politics** — Top political stories of the day
- **Sports** — Track specific players (default: Deni Avdija, Messi)
- **TV Shows** — Updates on shows you watch
- **Celebrity news** — Follow specific celebrities
- **Daily surprise** — Rotating: quote, joke, trivia, fun fact, today-in-history, riddle
- **Dark/Light mode** — With smooth transitions
- **Fully responsive** — Mobile, tablet, desktop
- **Three-level content** — Dashboard card → Hebrew summary → Original source
- **Every item has a source link** — Full transparency, verify anything

## Quick Start

### 1. Fork this repo

### 2. Edit `config.yaml`

```yaml
name: "Your Name"
weather:
  city: "Your City"
  country: "IL"
sections:
  sports:
    players:
      - "Your Player 1"
      - "Your Player 2"
  shows:
    list:
      - "Your Show 1"
      - "Your Show 2"
  celebrities:
    list:
      - "Celebrity Name"
```

### 3. Set up Claude Code scheduled tasks

The dashboard is generated by Claude Code running twice daily. Two scheduled tasks handle this:

- **Morning edition** (06:00) — Reads `scripts/generate-edition.md` and generates `docs/index.html`
- **Evening edition** (18:00) — Same process, different content focus

## How It Works

1. Claude Code scheduled task runs at 06:00 and 18:00
2. Searches the web for latest news on your configured topics
3. Translates and summarizes everything to Hebrew
4. Generates static HTML with real data
5. Pushes to GitHub → GitHub Pages serves the dashboard

## Customization

Everything is in `config.yaml`:
- **name** — Your name for greetings
- **city** — For weather and clothing recommendations
- **sections** — Enable/disable any section
- **sports.players** — Which athletes to track
- **shows.list** — Which TV shows to follow
- **celebrities** — Which celebrities to follow
- **schedule** — Morning and evening update times
